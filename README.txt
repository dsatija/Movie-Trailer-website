Movie Trailer Website project 

The "Best Movies website" is a website project wherein a server-side code is written 
to store a list of our favorite movies, including their poster image,storyline and a movie trailer URL.
The web page allows visitors to review their movies and watch the trailers.

Installation:

The project has been tested on  Python Version 2.7.10 and latest IE and Chrome browsers.
Unzip the Movies folder and place it in Python project-work location. 
Usage :

Ensure the path variable is set.
To run the project manually ,go to IDLE and run the module by name :
entertainment_centre.py
and it should open a web page titled Best Movies.
